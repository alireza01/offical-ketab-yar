# Quick Sound Setup (2 Minutes)

## Option 1: Download Free Sound (Recommended)

### Step 1: Visit Freesound
Go to: https://freesound.org/search/?q=marker+stroke

### Step 2: Find a Good Sound
Look for:
- Short duration (0.2-0.4 seconds)
- "Marker", "pen", or "swoosh" sounds
- Clean, simple audio

### Step 3: Download
- Click on a sound you like
- Click "Download" button
- Save the file

### Step 4: Convert to MP3 (if needed)
If the file is not MP3:
- Use online converter: https://cloudconvert.com/to/mp3
- Upload your file
- Download as MP3

### Step 5: Rename and Place
- Rename file to: `highlight.mp3`
- Move to: `public/sounds/highlight.mp3`
- Done! âœ…

## Option 3: No Sound (Silent Mode)

Don't want sound? No problem!

The app works perfectly without the sound file. It will just skip the audio and work silently. No errors, no issues.

---

## Testing

After adding the sound:
1. Open the reader
2. Select some text
3. Click a highlight color
4. You should hear a soft "swoosh" sound

If you don't hear anything:
- Check browser console for errors
- Verify file is named exactly `highlight.mp3`
- Try refreshing the page
- Check browser volume settings
