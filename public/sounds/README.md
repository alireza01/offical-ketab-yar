# Sound Effects for Ketab-Yar Reader

## Highlight Sound Effect

**File**: `highlight.mp3`

### Requirements:
- **Duration**: 0.2-0.4 seconds (short and subtle)
- **Type**: Soft "swoosh" or "marker stroke" sound
- **Volume**: Quiet and non-intrusive
- **Format**: MP3 (for browser compatibility)

### How to Create:

#### Option 1: Use Free Sound Libraries
1. Visit [Freesound.org](https://freesound.org)
2. Search for: "marker", "pen stroke", "swoosh", or "highlight"
3. Download a short, soft sound effect
4. Convert to MP3 if needed
5. Save as `highlight.mp3` in this directory

#### Option 2: Generate with AI
1. Use [ElevenLabs Sound Effects](https://elevenlabs.io/sound-effects)
2. Prompt: "soft marker pen stroke on paper, quick swoosh"
3. Download and save as `highlight.mp3`

#### Option 3: Record Your Own
1. Use a real highlighter on paper
2. Record with your phone
3. Trim to 0.2-0.4 seconds
4. Export as MP3

### Recommended Free Sounds:
- "Marker Stroke" by InspectorJ (freesound.org)
- "Pen Swipe" by plasterbrain (freesound.org)
- "Quick Swoosh" by qubodup (freesound.org)

### Fallback:
If no sound file is present, the app will work silently (no errors).
The sound is optional and enhances the experience but isn't required.

---

**Note**: Make sure the file is named exactly `highlight.mp3` and placed in this directory.
